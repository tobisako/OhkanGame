# とある改造方法

Ｑ：なぜ、スイッチ一つで切り替える様にしないのか？
Ａ：大人の事情です。あんまし簡単に見れる様にするのを避ける為です。

※本件の概要は、「README.me」をご覧ください。



◆1.ファイルを移動する。

「bin_sonota」フォルダ内にある全ファイルを、
「bin」フォルダへ移動させてください。


◆2.プログラムを改造する。

下記手順通り、プログラムを修正して下さい。

※下記「Ｂ」をコメントアウトして、上記「Ａ」のコメントアウトを外して有効にして下さい。
